Detailed Description

This file contains the code for creating and managing a basic window using CSFML. This one is using a struct to store all the window data.

Note
You do not need to include the CSFML library in the source file, as it is already included in the header file.
You do not need to include the stdlib.h library in the source file, as it is already included in the header file.
You can use the typedef struct directly in the source file, but it is recommended to put it in a header file.
You will need to typedef first your struct in the header file. Else it will not compile.

window_t* init ( void  )

Initializes a window, and returns a pointer to it.

This function allocates memory for a window_t struct, initializes its mode and event members, and returns a pointer to it.

window_t *init(void)
{
// Allocate memory for the window struct
window_t *window = malloc(sizeof(window_t));
// Initialize the window's mode and event members
window->mode = (sfVideoMode){800, 600, 32};
window->window = NULL;
window->event = (sfEvent){0};
// Return a pointer to the window
return (window);
}

int main ( void  )

The main function.

This function initializes a window, creates a third window using CSFML, and manages its events and rendering.

Note
The given struct can be changed, it doesn't need to be this one, you can put yours and personalize it like you want.
int main(void)
{
// Initialize window data
window_t *window_data = init();
sfVideoMode mode = window_data->mode;
sfRenderWindow *window = window_data->window;
sfEvent event = window_data->event;
// Create the third window
window = sfRenderWindow_create(mode, "My third window", sfResize | sfClose, NULL);
// Check if the window was created successfully and return 84 if not
if (!window)
return (84);
// Principal loop that checks if the window is open
while (sfRenderWindow_isOpen(window)){
// Secondary loop that poll the events
while (sfRenderWindow_pollEvent(window, &event)){
// If the user closes the window, close the window
if (event.type == sfEvtClosed)
sfRenderWindow_close(window);
}
// Clear the window with a black color
sfRenderWindow_clear(window, sfBlack);
// Display the window
sfRenderWindow_display(window);
}
// Clean up and return
sfRenderWindow_destroy(window);
// Free the window data
free(window_data);
return (0);
}
